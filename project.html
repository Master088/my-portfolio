Web-based Resource Management System Using QR Codes

This system is used to track who comes in and out of the IT reading center. This is a web-based frontend system for recording using QR codes, as well as a sub-system for the Resource Management System. This system was built using HTML, CSS, Bootstrap, React JS, and React Idle Timer.
